PITSTOP 1 OF 5 · SITE VISIT · 7 DAYS

Hotspot Assessment

Why this pitstop

Visit and assess the proposed hotspot before committing. Confirm vehicle access, beneficiary population, serving time window, and no overlap with existing DPs.

1.
Location visited — beneficiary population estimated
✓Hotspot Field Visit
2.
Hotspot type confirmed — bus stand / railway / naka / shelter / hospital / other
🎙Log Hotspot Type Confirmation
3.
TATA Ace access and parking confirmed — route driveable from existing stops
🎙Log Vehicle Access Check
4.
Optimal serving time window assessed for this location
🎙Log Serving Window Assessment
5.
Overlap with existing DPs checked — no cannibalisation
🎙Log DP Overlap Check
6.
Community / local authority informed of plans
🎙Log Community Notification
PITSTOP 2 OF 5 · MEETING · 21 DAYS

DP Personnel Identification & Agreement

Why this pitstop

Find a DP person via Sampark/APSA community networks. Brief them fully on the role — timing, kit handling, hygiene, daily reporting. Get a signed agreement.

1.
DP personnel candidate identified via Sampark/APSA network
🎙Identify DP Personnel Candidate
2.
Candidate briefed on role — timing, kit, hygiene, reporting
✓DP Personnel Briefing Meeting
3.
Agreement signed
📎Upload Signed DP Agreement
4.
Emergency contact and backup person identified
🎙Log Emergency Contact Details
PITSTOP 3 OF 5 · TRAINING · 35 DAYS

DP Personnel Training

Why this pitstop

Train the new DP person at the actual hotspot. Cover all protocols and do a mock setup with the full kit before the first live distribution.

1.
Queue management protocol trained
✓DP Training Session at Hotspot
2.
Hygiene protocol trained
🎙Confirm Hygiene Protocol Trained
3.
Kit setup and teardown trained — full mock with actual kit
✓Mock Setup at Hotspot
4.
Daily reporting and loss documentation trained
🎙Confirm Reporting Training
PITSTOP 4 OF 5 · REVIEW · 42 DAYS

Route Integration

Why this pitstop

Insert the new DP into the route sequence. Recalculate timing across all stops to confirm everyone is still served within the window. Brief driver and update vehicle bay plan.

1.
New DP inserted into route sequence — FILO loading order updated
📎Upload Updated Route Plan
2.
End-to-end timing recalculated — all DPs still within window
📎Upload Recalculated Route Timing
3.
Driver briefed on new stop
✓Route Update Meeting with Driver
4.
Kit assembled and numbered for new DP
📎Upload New DP Kit Inventory
5.
Vehicle bay layout updated — new kit confirmed to fit
📎Upload Updated Bay Layout

4.
DP personnel debrief done
✓Post Day-1 Debrief with DP Person
5.
DP marked active in programme tracker
